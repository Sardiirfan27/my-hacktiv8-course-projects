Website apa yang paling ingin kamu buat? Tentang apa?

Selama ini saya telah membuat beberapa website untuk keperluan organisasi non-profit, beberapa proyek yang menghasilkan uang, hingga membangun produk di beberapa start-up. Namun, ironisnya, justru website pribadi saya sendiri tidak sempat saya bangun ): padahal saya telah membeli domain dan vps sejak dua tahun yang lalu, tapi website saya sampai sekarang masih coming-soon. Ironi bukan ? Haha. Mungkin beberapa diantara kalian pasti juga mengalaminya. Haha.
Jadi, ya, Saya ingin membuat website yang berisi profil, portofolio, tulisan dan kontak saya. Saya ingin meningkatkan self-branding saya, karena ternyata hal tersebut sangatlah penting untuk memnunjang karir saya kedepan.

Hasil atau informasi apa yang paling ingin kamu sampaikan?

Sejujurnya, Saya sangat terinspirasi dari website mas Adham Dannaway disana di menggambar dengan jelas siapa dirinya, apa ketrampilannya, proyek yang pernah dibuat, tulisan, fakta-fakta menarik tentang dirinya , website / majalah yang memberitakan tentang dirinya serta tidak lupa kontak agar orang dapat menghubunginya dengan mudah.
Saya akan membuat hal serupa, namun dengan cita rasa saya sendiri.

Bagaimana kira-kira kamu memproses pembuatan website dari awal hingga akhir?

Proses pembuatan website pribadi saya akan dimulai dari riset kecil-kecilan tentang apa sih informasi yang dicari orang tentang diri saya ?
Setelah mengetahui hal tersebut, saya bisa membuat wireframe, sitemap dan design guideline untuk proyek ini sebelum mengtransformasikannya menjadi website.
Tech stack yang akan saya gunakan :

  • Back-end : Menggunakan PHP dengan framework Laravel versi 5.3
  • Front-end : Rencana saya akan menggunakan Bootstrap 3/4 dan jQuery. Walaupun, sebenarnya saya ingin sekali menggunakan Vue.js, saya urungkan karena untuk website ini belum membuthkan komponen dan logika program yang komplek, serta saya belum menemukan solusi terbaik perihal SEO pada aplikasi SPA (Single Page Application)
  • Versioining Control : Saya percayakan pada Git dan Bitbucket / Github serta menggunakan Semantic Versioning 2.0.0
  • Blog : Saya memasrahkan tulisan saya kepada Medium karena ini merupakan platform bloging untuk membaca dan menulis yang sangat keren menurut saya. Namun, dengan custom domain saya sendiri. Rencana akan di blog.dikyarga.com
  • Server : Saya menggunakan VPS dari Digital Ocean . Tentu dengan spesifikasi paling minimal disana. Haha
Guideline :
  • Search Engine Optimation in mind, segala sesuatunya harus mendukung SEO agar website ini mudah ditemukan orang di Internet
  • Desain UI dan UX akan mengikuti Material Design yang dibuat oleh Google
